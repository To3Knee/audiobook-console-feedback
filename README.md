# Audiobook Console — Feedback

Public hub for **Audiobook Console** beta feedback, feature requests, and
release announcements.

**The app source lives in a separate (currently private) repo.** This repo
exists so testers can file ideas, ask questions, and read announcements
without needing access to the codebase.

## What this is for

- **Bug reports** — open an Issue. Include device, iOS version, build,
  and steps to reproduce.
- **Feature requests / ideas** — open a Discussion in the **Ideas**
  category.
- **Questions** — open a Discussion in the **Q&A** category.
- **Release notes** — posted to the **Announcements** category by
  the maintainer.

## What this is not

- This repo has no code. Pull requests will be politely closed — the
  app source is private and managed separately.
- This is not a public support channel. Audiobook Console is a
  **friends-and-family TestFlight beta**, not a shipped product.

## More about Audiobook Console

Audiobook Console is a self-hosted dashboard for your own Audible
library. It runs in Docker on your hardware, talks to Audible with your
account, and keeps a Git-backed history of the books you already own.
The iOS app is a thin Capacitor wrapper around the same PWA that runs
in your browser.
